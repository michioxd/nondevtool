nondevtool aka Anti-devtools

## How to use
Just include nondevtool.js to your page and try open devtool ehehe :))

## Example
```html
<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8">
    <title>Test</title>
  </head>
  <body>
    <h1>Test</h1>
    <script src="./nondevtool.js"></script>
  </body>
</html>
```
