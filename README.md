# nondevtool
Anti-devtools

## Demo
See the demo <a href="https://nondevtool.michio.ga">here</a>
## How to use
Just include this file JS to your page and try open devtool ehehe :))

## Example
```html
<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8">
    <title>Test</title>
  </head>
  <body>
    <h1>Test</h1>
    <script src="./nondevtool.js"></script>
  </body>
</html>
```
